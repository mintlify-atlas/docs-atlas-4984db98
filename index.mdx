Whether you're working with plain JavaScript, TypeScript syntax, or JSDoc-flavored TypeScript, it provides ready-made presets and fine-grained rule control.

## Get started in 3 steps

<Steps>
  <Step title="Install the plugin">
    Add eslint-plugin-jsdoc to your project as a dev dependency.

    ```bash
    npm install --save-dev eslint-plugin-jsdoc
    ```
  </Step>
  <Step title="Add a preset config">
    Use a recommended preset in your ESLint flat config file.

    ```js eslint.config.js
    import jsdoc from 'eslint-plugin-jsdoc';

    export default [
      jsdoc.configs['flat/recommended'],
    ];
    ```
  </Step>
  <Step title="Run ESLint">
    Run ESLint to start catching JSDoc issues in your code.

    ```bash
    npx eslint .
    ```
  </Step>
</Steps>

<Note>
  eslint-plugin-jsdoc supports ESLint 7 through 10, with full flat config support for ESLint 9+. Node.js 20.19+, 22.13+, or 24+ is required.
</Note>
